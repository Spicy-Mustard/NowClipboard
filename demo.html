现代剪贴板工具库 — 交互演示

v1.1.8
环境检测： 复制 … 剪切 … 读取 … 协议 …

📋复制文本

使用 NowClipboard.copy() 将任意文本写入剪贴板
点击按钮查看结果

📥读取文本

使用 NowClipboard.read() 从剪贴板读取文本
点击按钮查看结果

👂剪贴板监听

使用 NowClipboard.onChange() 轮询监听剪贴板内容变化
点击"开始监听"后，复制/粘贴内容会在这里显示

🖼️图片操作

使用 NowClipboard.copyImage() 复制图片到剪贴板
点击按钮查看结果

🔐权限与协议

查询剪贴板权限状态和当前页面协议
点击按钮查看结果

📊特性检测

使用 NowClipboard.checkSupport() 检测支持的剪贴板功能
点击按钮查看结果

✏️写入文本（write）

使用 NowClipboard.write() 写入文本（copy 的语义别名，支持 Node.js）
点击按钮查看结果

🎨写入图片（writeImage）

使用 NowClipboard.writeImage() 写入图片到剪贴板（浏览器 + Node.js）
点击按钮查看结果

📦多格式写入（writeFormats）

使用 NowClipboard.writeFormats() 一次写入多种 MIME 类型
点击按钮查看结果

📜剪贴板历史（History）

使用 NowClipboard.History 自动记录剪贴板变更，支持搜索和持久化
点击"开始记录"后，复制的内容会自动记录

🔗跨标签页同步（onSync）

使用 NowClipboard.onSync() 基于 BroadcastChannel 在标签页间同步复制操作
开启后，在其他标签页复制的操作会同步到此处

💡使用提示

推荐使用本地服务器打开此页面，避免 file:// 协议的权限限制： 
npx serve .
# 然后访问 http://localhost:3000/demo.html
原因：浏览器不会为 file:// 协议持久化剪贴板权限，每次读取都会重新弹出授权弹窗。 通过 HTTP/HTTPS 访问时，权限只需授权一次。
